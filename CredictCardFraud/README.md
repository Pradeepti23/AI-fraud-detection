# Credit Card Fraud Detection System

## Overview

A Machine Learning based Credit Card Fraud Detection System built using:

- Python
- Flask
- Random Forest
- XGBoost
- SQLite
- HTML
- CSS
- JavaScript

---

## Features

- Fraud Prediction
- Real-time Risk Analysis
- Dashboard Analytics
- Transaction History
- Random Forest Model
- XGBoost Model
- SQLite Database
- Responsive UI

---

## Dataset Features

- Transaction Amount
- Transaction Type
- Merchant Category
- Transaction Time
- Customer Age
- Previous Fraud History
- New Device Used
- International Transaction
- Location Match
- Transactions Today
